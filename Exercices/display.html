Displays

Les Displays

Display : Block

Cette div est un élément en Block. Il occupe toute la largeur de l'écran

Exemple de balises de type 'Block':

  • div
  • p
  • h1,h2,h3,h4
  • li
  • main
  • ul

Display : Inline

Les éléments en Inline ne forcent pas le retour à la ligne, il prend simplement la place dont il a besoin.

Attention !! Les éléments de types inmine ne sont pas comme les types block, leur hauteur et leur largeur ne sont délémités QUE par leur contenu

Exemples de balises de type 'Inline'

  • balise a (liens)
  • image
  • vidéos
  • input
  • audio
  • label
  • span

Display : Inline-Block

Inline-Block : c'est un type inline qui conserve les propriétes du type block, il peut donc etre modifié en hauteur et en largeur comme n'importe quel type de block

Display : Flex

Flex : Les éléments dans le conteneur s'alignent en ligne et se répartissent l'espace disponible

Display : Grid

Grid : Avec grid, on peut définir une grille avec des lignes et des colonnes

Display : None

None : Le dsisplay none cache tout simplement l'élément.
